# Prática em EVA — Lips Sense Technique

## Movimentos para Bordas
> Aula: 06Prática EVA_Sub_01_Movimentos para Bordas

**Configuração do aparelho para esta prática:**
- Agulha: uma ponta Nano 2, 0,25mm (ou 0,30mm)
- Velocidade: 120 (no AMIA)
- Trabalha só a pontinha da agulha

**Contorno — três opções de movimento:**

1. **Linha contínua (bidirecional BVP)** — movimento lento, vai e volta. Para lábio bem definido, contorno sólido. É o que o James usa mais para quando quer linha muito definida.

2. **Cachemir** — movimento rápido e longo, cria linha pontilhada. Para lábio mais neutro, natural, com borda mais suave. Não satura tanto. Usa quando quer só transferir o desenho para a pele sem chamar atenção.

3. **Pontilhismo** — só a pontinha, movimentos curtos, sem arraste. Aplica de maneira mais irregular. Baixíssima saturação. Para quando quer acabamento bem discreto.

**Importante:** sempre começa pelas comissuras, independente da técnica escolhida.

## Movimentos Base ABC
> Aula: 06Prática EVA_Sub_02_Movimentos Base ABC

**Os dois movimentos fundamentais:**
- Puxar para o coração (para si)
- Empurrar para fora (contra o coração)

Conforme você aumenta a velocidade e diminui o espaçamento entre as linhas, você passa de cachemir para shadow.

**Cachemir** → linhas levemente separadas, não há sobreposição.
**Shadow** → linhas muito próximas, encostando uma na outra, com sobreposição de pontos (vai e volta cria pontos sobrepostos).

A diferença visual entre os três efeitos (cachemir, shadow intermediário, shadow denso) é apenas o espaçamento entre as linhas.

**Nove variações de movimento:**
Combinando direção (para si / para cima / para fora) com intensidade (cachemir / shadow / shadow+sobreposição) = 9 variações. O nome padrão é A, B e C.

**O mais usado para lábios:** variação 3A, 3B e 3C (puxando para si, shadow). É a variação mais usual para desenvolver as características de preenchimento labial.

**Atenção ao movimento C:** ele rasga a pele se feito de maneira errada (vai e volta mais agressivo). Os pontinhos do C têm menos pigmento dentro — significam mais trauma. O B tem mais pigmento dentro dos pontos — melhor implantação. **Não use o C nas comissuras** — o vai e volta rápido tem muita chance de sair do desenho.

## Movimentos Aplicados na Modelo
> Aula: 06Prática EVA_Sub_03_Movimentos Aplicados

**Sequência de aplicação:**

1. Após o contorno, imagine os quadrantes. Divida o lábio.
2. Comece pelas **comissuras** — da linha para dentro, da linha para dentro. Vai para o outro lado, da linha para dentro. Faz as comissuras primeiro dos dois lados antes de avançar.
3. Depois das comissuras, comece o movimento de varredura (MSB — Movimento em Sentido à Borda): horizontal em relação à borda, subindo e descendo.

**Problemas comuns de iniciantes:**

- **Entrada pesada no shadow:** o primeiro toque da agulha na pele muito pesado cria uma marca de entrada e a saída fica esfumada. Mancha.
- **Linha fantasma:** medo de chegar na linha de contorno — fica um espaço branco entre o preenchimento e a borda. Use pontilhismo para o acabamento nessa área se tiver com medo.
- **Sair um pouquinho para fora:** uma passada não pigmenta. Limpa e continua. Fique em paz.

**Por que fazer duas direções?**

Quando você faz só para uma direção, cria uma "fibra." Quando você cruza com a outra direção, quebra essa fibra e o trabalho fica muito mais homogêneo.

**Região da área úmida (parte mais centralizada/interna do lábio):**

É abaulada — quando você olha de lado parece um peixinho. Nessa região, trabalhe **sempre em diagonal** — não movimentos horizontais, porque a superfície é abaulada e os movimentos horizontais não conseguem trabalhar direito. Diagonal, diagonal, diagonal.

**Região do arco do cupido:**

É difícil porque você tem que trabalhar sentidos opostos: para cá e para lá. Quando trabalha para um lado, beneficia duas bordas; quando trabalha para o outro, beneficia as outras duas bordas. Não negligencie essa região — sempre fica mais clara se você não trabalhar ela especificamente.

## Exercício de Varredura — Quadrantes
> Aula: 07Aula Prática EVA 2_Sub_01_Exercício de Varredura

**Exercício do quadrante:**

Faz o movimento A (puxando para si) horizontal, varrendo todo o bloco. Depois faz com o B (puxando para cima). As linhas têm que ser paralelas e justapostas.

**Regra de avanço:** avance o mínimo possível — de pouquinho em pouquinho. Quanto menos avança antes de varrer, menor a probabilidade de manchar.

**O treino mais importante de todo o curso:**

Se você não estiver perfeito no exercício de varredura do quadrante, não vai estar perfeito na boca. A boca é exatamente isso: você tem que trabalhar a borda longa e a borda curta do quadrante.

**Degradê vs. sólido:**
- **Degradê:** começa na borda e vai deixando a mão leve conforme entra — cria um esfumado natural.
- **Sólido:** começa na borda, vai até o fundo, volta. Mantra: borda, borda, dentro, dentro.

**Atenção à borda lateral:** no treino do quadrante, as duas laterais têm que ficar perfeitas. Pratique os dois lados separadamente.

Sequência de treino do quadrante:
- A começando de cima para baixo
- A começando de baixo para cima
- B começando de cima para baixo
- B começando de baixo para cima

## Demonstração com Dermógrafo
> Aula: 07Aula Prática EVA 2_Sub_02_Demonstração do Exercício com o Dermógrafo

**Efeitos visuais comparados:**
- Trabalho todo no pontilhismo → efeito visual
- Trabalho todo no shadow → visualmente quase igual ao pontilhismo na foto; diferença está na execução
- Degradê de borda → efeito mais suave nas bordas
- Preenchimento sólido → mais saturado
- Preenchimento com borda muito definida → não é o favorito do James para foto

**Movimento circular** → fica visivelmente mais escuro na região. Use só com agulhas circulares.

**Aplicação progressiva com o dermógrafo:**

Começa varrendo o primeiro bloco horizontal, vai descendo paulatinamente, repassando um pouco sobre a camada anterior. Tamanho do bloco: 4 quadrantes (não maior — nenhuma boca tem essa largura toda de uma vez).

**O treino mais importante:** quadrante, com foco total na borda. Se a borda não estiver perfeita no treino, não vai estar perfeita na boca. Pratique A de cima para baixo, A de baixo para cima, B de cima para baixo, B de baixo para cima — isso é a boca real.

**Cachemir vs. linhas próximas:**

O cachemir machuca menos e pigmenta mais do que parece. Se você partir para o cachemir (linhas bem próximas), pigmenta mais e machuca menos do que o shadow — boa estratégia para peles mais sensíveis.
